# Ejecutar en desarrollo

1. clonar el repositorio
2. ejecutar
 
Nota: si estas en un entorno restringido puedes intentar usar
```
  npx al inicio de npm
  npx npm install
```
```
npm install
```

3. Tener nest cli instalado

```
npm i -g @nestjs/cli
```
4. levantar la base de datos
```
docker-compose up -d
```
## Stack usado

* MongoDB
* Nest
